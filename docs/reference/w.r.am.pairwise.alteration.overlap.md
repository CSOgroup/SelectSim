# Compute null weighted overlap matrix

Compute null weighted overlap matrix

## Usage

``` r
w.r.am.pairwise.alteration.overlap(null, W, n.permut, n.cores = 1)
```

## Arguments

- null:

  The null model (list of simulated binary matrices)

- W:

  The weight matrix

- n.permut:

  The number of permutation steps

- n.cores:

  The number of cores

## Value

weighted overlap summed across permutations
