# Extract the backgroun distribution

Create an Alteration Landscape (AL) object which contains gams and
mutation burden of samples of associated gams.

## Usage

``` r
overlap_pair_extract(gene1, gene2, obj)
```

## Arguments

- gene1:

  gene1

- gene2:

  gene2

- obj:

  selectX obj

## Value

a backgorund distribution vector.
