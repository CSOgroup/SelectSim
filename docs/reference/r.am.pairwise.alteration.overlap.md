# Compute null overlap matrix

Compute null overlap matrix

## Usage

``` r
r.am.pairwise.alteration.overlap(null, n.permut, n.cores = 1)
```

## Arguments

- null:

  The null model (list of simulated binary matrices)

- n.permut:

  The number of permutation steps

- n.cores:

  The number of cores

## Value

overlap the overlap summed across permutations
